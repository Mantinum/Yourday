# Yourday

Monorepo pour l'application Yourday. Le socle est prêt pour les futures implémentations.

## Installation future

```bash
pnpm install
pnpm prisma migrate dev
```

## Scripts

- `pnpm dev` – démarrage des apps en développement (à définir)
- `pnpm build` – build
- `pnpm test` – tests

Ces commandes sont pour l'instant des placeholders.
