# API

## Health
- `GET /health`

## Recipients
- `POST /recipients`
- `GET /recipients`
- `GET /recipients/:id`
- `PATCH /recipients/:id`
- `DELETE /recipients/:id`

## Events
- `POST /events`
- `GET /events`
- `GET /events/:id`
- `PATCH /events/:id`
- `DELETE /events/:id`

## Recommendations

## Orders

## Webhooks
